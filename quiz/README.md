# Quiz motoryzacyjny — Vue 3

Gotowy projekt aplikacji quizowej wykonany w Vue 3 + Vite.

## Funkcje

- 3 ekrany: start, pytanie, wynik
- 5 pytań o motoryzacji
- 4 odpowiedzi do każdego pytania
- podświetlanie poprawnej i błędnej odpowiedzi
- blokada zmiany odpowiedzi po kliknięciu
- pasek postępu i licznik pytań
- wynik procentowy i komunikat końcowy
- reset quizu
- przełączanie między **dark mode** i **light mode**
- zapamiętywanie wybranego motywu w `localStorage`

## Uruchomienie

```bash
npm install
npm run dev
```

## Build produkcyjny

```bash
npm run build
```

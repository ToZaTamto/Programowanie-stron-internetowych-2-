# Kalkulator BMI — Cosmic Mode

Projekt wykonany w Vue 3 + Vite z użyciem Composition API i składni `<script setup>`.

## Uruchomienie

```bash
npm install
npm run dev
```

## Build produkcyjny

```bash
npm run build
```

## Zrealizowane wymagania

- 2 pola input z `v-model`
- obliczanie BMI przez `computed()`
- wynik aktualizowany na bieżąco bez przycisku
- kategoria BMI wyświetlana warunkowo
- różne kolory kategorii
- brak wyświetlania wyniku dla pustych lub błędnych danych
- osobny komponent w osobnym pliku `.vue`
- dark / black mode
- styl kosmiczny
